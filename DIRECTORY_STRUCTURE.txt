FLUXO VISUAL: Da Edição ao Deploy Automático

LOCAL (seu PC)
   ↓
   git edit → git add → git commit → git push origin main
   ↓
GITHUB (repositório remoto)
   ↓
   GitHub Actions dispara (workflow: deploy.yml)
   ↓
   ✓ Valida structure (plugin.json válido?)
   ✓ Calcula versão (feat/fix/docs)
   ✓ Atualiza VERSION
   ✓ Cria git tag (v0.2.0)
   ✓ Empacota plugin-v0.2.0.zip
   ✓ Publica Release
   ↓
RELEASE (disponível para consumo)
   ↓
   URL: https://github.com/seu-usuario/chiapetti-plugin-legal/releases/latest/download/plugin.zip
   ↓
CLAUDE COWORK
   ↓
   Plugin se atualiza automaticamente (próxima inicialização)

EDITAR ARQUIVO: Passos Concretos

1. Abra editor (VS Code, Notepad++, etc.)
2. Abra arquivo:
   C:\Users\seu-usuario\chiapetti-plugin-legal\plugin\skills\analise-processo\SKILL.md

3. Edite conteúdo

4. Salve (Ctrl+S)

5. Abra terminal/PowerShell em: C:\Users\seu-usuario\chiapetti-plugin-legal\

6. Rode:
   git add plugin/skills/analise-processo/SKILL.md
   git commit -m "fix: improve risk assessment logic"
   git push origin main

7. ✓ Deploy automático começa
